Android Development: Budgeting App

About this project

Managing my finances effectively has always been a priority - with this project I envisioned an app that not only tracks expenses but also does so with an appealing visual design and ease of use. My goal was to create something I can use for myself.

Key Features:

  • Secure user registration with email verification.
  • Custom budgets for different expense categories, short-term or long-term.
  • Easily log and manage daily expenses and categorize them.
  • Receive notifications upon exceeding the budget.
  • User-Friendly Design, easy-to-use interface.

Code Technologies:

  • Android Studio
  • Java
  • XML
  • Firebase
