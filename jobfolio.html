Web Development: JobFolio

About this project

JobFolio is a platform designed for career starters, offering the tools to create and manage a powerful portfolio that highlights personal details, projects, and skills. In this project, JobFolio focuses primarily on the portfolio creation, providing an accessible solution for career starters. It simplifies the process, allowing users to create and manage an appealing portfolio by easily filling out forms and uploading images. In addition, the entire project was documented in GitHub Pages.

Key Features:

  • Account creation with distinct views based on user roles.
  • Profile management, including name, current title, description, and profile picture.
  • Skill and language management.
  • Project addition and management.
  • Placeholder pages for job search and platform homepage.

Code Technologies:

  • Python
  • HTML & CSS
  • SQLite
  • Flask
  • GitHub Pages
