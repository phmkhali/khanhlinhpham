Machine Learning: Passenger Satisfaction

About this project

This is a simple Machine Learning project that revolves around the binary classification task of predicting passenger satisfaction with airlines. I've utilized a Kaggle dataset, which gathers feedback from passenger surveys. To accomplish this, I built and evaluated four distinct models to identify the most suitable one for the task at hand.

Project Content:

  • Exploratory Data Analysis
  • Data Preparation
  • Model Building
  • Model Evaluation
  • Potential Business Application

Code Technologies:
